Politique de confidentialité / Privacy Policy

Dernière mise à jour / Last updated: 17 janvier 2026

Français

1. Introduction

Jass Points ("nous", "notre", "l'application") est une application de comptage de points pour le jeu de cartes suisse Jass. Nous respectons votre vie privée et nous nous engageons à protéger vos données personnelles.

2. Données collectées

2.1 Données locales (stockées uniquement sur votre appareil)

  • Parties de Jass : Scores, noms d'équipes, historique des parties
  • Préférences de l'application : Thème, paramètres de jeu
  • Préréglages personnalisés : Configurations de parties sauvegardées
  • État de la version Pro : Statut d'achat de la version Pro (stocké localement)

Ces données restent strictement sur votre appareil et ne sont jamais transmises à nos serveurs.

2.2 Données collectées lors de rapports de bugs ou suggestions (optionnel)

Lorsque vous choisissez d'envoyer un rapport de bug ou une suggestion d'amélioration, nous collectons :

  • Titre et description fournis par vous
  • Adresse email (si vous choisissez de la fournir - optionnel)
  • Informations techniques automatiques :
    • Version de l'application
    • Modèle et fabricant de l'appareil
    • Version Android
    • Langue système

Ces données sont collectées uniquement si vous choisissez d'envoyer un rapport et sont stockées dans Firebase Firestore.

2.3 Données analytiques (Firebase Analytics)

Nous utilisons Firebase Analytics pour comprendre comment l'application est utilisée :

  • Statistiques d'utilisation anonymes
  • Données de crash (Firebase Crashlytics) pour améliorer la stabilité
  • Données agrégées sur les fonctionnalités utilisées
  • Identifiants d'appareil anonymes (Firebase Installation ID)

Ces données sont anonymisées et ne permettent pas de vous identifier personnellement.

2.4 Données de paiement (version Pro)

Pour les achats in-app (version Pro), Google Play gère les transactions. Nous ne collectons ni ne stockons vos informations de paiement. Seul l'état de votre achat (Pro activé ou non) est sauvegardé localement sur votre appareil.

3. Utilisation des données

Les données collectées sont utilisées pour :

  • Fonctionnement de l'application : Sauvegarder vos parties et préférences localement
  • Amélioration : Analyser les bugs signalés et implémenter des améliorations
  • Stabilité : Détecter et corriger les plantages via Crashlytics
  • Statistiques : Comprendre l'utilisation pour améliorer l'expérience utilisateur

4. Partage des données

Nous ne vendons ni ne partageons vos données personnelles avec des tiers, sauf :

  • Google Firebase : Pour le stockage des rapports de bugs/suggestions et l'analyse anonyme (soumis à la politique de confidentialité de Google)
  • Google Play : Pour les achats in-app (soumis à la politique de confidentialité de Google)

5. Sécurité des données

  • Les données locales sont stockées sur votre appareil avec les mécanismes de sécurité Android
  • Les communications avec Firebase utilisent le chiffrement HTTPS/TLS
  • Les rapports de bugs sont stockés sur des serveurs Firebase sécurisés avec chiffrement au repos

6. Vos droits

Vous avez le droit de :

  • Accéder à vos données : Consultez l'historique dans l'application
  • Supprimer vos données : Utilisez la fonction "Supprimer tout l'historique" dans l'application
  • Retirer votre consentement : Désinstallez l'application pour supprimer toutes les données locales

Pour supprimer vos données stockées sur nos serveurs (rapports de bugs/suggestions déjà envoyés), veuillez nous envoyer un email à kijunpulz@gmail.com avec votre demande de suppression.

7. Données des enfants

L'application ne cible pas les enfants de moins de 13 ans et ne collecte pas sciemment de données d'enfants.

8. Rétention des données

  • Données locales : Conservées indéfiniment sur votre appareil jusqu'à suppression ou désinstallation
  • Données Analytics/Crash : Conservées selon les paramètres par défaut de Firebase (60 jours pour Analytics, 90 jours pour Crashlytics)
  • Rapports de bugs : Conservés jusqu'à 2 ans à des fins d'amélioration

9. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité. Les modifications seront publiées sur cette page avec une nouvelle date de mise à jour.

10. Contact

Pour toute question concernant cette politique de confidentialité :

  • Email : kijunpulz@gmail.com
  • Depuis l'application : Menu → Signaler un bug

English

1. Introduction

Jass Points ("we", "our", "the application") is a score tracking application for the Swiss card game Jass. We respect your privacy and are committed to protecting your personal data.

2. Data Collected

2.1 Local Data (stored only on your device)

  • Jass Games: Scores, team names, game history
  • App Preferences: Theme, game settings
  • Custom Presets: Saved game configurations
  • Pro Version Status: Purchase status of Pro version (stored locally)

This data remains strictly on your device and is never transmitted to our servers.

2.2 Data Collected for Bug Reports or Suggestions (optional)

When you choose to send a bug report or improvement suggestion, we collect:

  • Title and description provided by you
  • Email address (if you choose to provide it - optional)
  • Automatic technical information:
    • App version
    • Device model and manufacturer
    • Android version
    • System language

This data is collected only if you choose to send a report and is stored in Firebase Firestore.

2.3 Analytics Data (Firebase Analytics)

We use Firebase Analytics to understand how the application is used:

  • Anonymous usage statistics
  • Crash data (Firebase Crashlytics) to improve stability
  • Aggregated data on features used
  • Anonymous device identifiers (Firebase Installation ID)

This data is anonymized and does not allow personal identification.

2.4 Payment Data (Pro version)

For in-app purchases (Pro version), Google Play handles transactions. We do not collect or store your payment information. Only your purchase status (Pro enabled or not) is saved locally on your device.

3. Data Usage

Collected data is used to:

  • Application Operation: Save your games and preferences locally
  • Improvement: Analyze reported bugs and implement improvements
  • Stability: Detect and fix crashes via Crashlytics
  • Statistics: Understand usage to improve user experience

4. Data Sharing

We do not sell or share your personal data with third parties, except:

  • Google Firebase: For bug report/suggestion storage and anonymous analytics (subject to Google's privacy policy)
  • Google Play: For in-app purchases (subject to Google's privacy policy)

5. Data Security

  • Local data is stored on your device with Android security mechanisms
  • Communications with Firebase use HTTPS/TLS encryption
  • Bug reports are stored on secure Firebase servers with encryption at rest

6. Your Rights

You have the right to:

  • Access your data: Check history in the app
  • Delete your data: Use the "Delete all history" function in the app
  • Withdraw consent: Uninstall the app to remove all local data

To delete your data stored on our servers (bug reports/suggestions already sent), please send us an email at kijunpulz@gmail.com with your deletion request.

7. Children's Data

The application does not target children under 13 and does not knowingly collect children's data.

8. Data Retention

  • Local data: Stored indefinitely on device until user deletes or uninstalls
  • Analytics/Crash data: Retained according to Firebase default retention (60 days for Analytics, 90 days for Crashlytics)
  • Bug reports: Retained for up to 2 years for improvement purposes

9. Changes to this Policy

We may update this privacy policy. Changes will be published on this page with a new update date.

10. Contact

For any questions regarding this privacy policy:

  • Email: kijunpulz@gmail.com
  • From the app: Menu → Report a Bug
